# Switching Your E-Commerce Store Integration from NextGen to Classic Version
To become back the Classic User, you will need to ensure that your API key version is in Classic. For the guideline that how to change API key version, you may refer to the steps below.

## Navigation Path
#### Login to EasyParcel website -> EasyConnect -> Installed Ecommerce Apps -> Select your ecommerce store -> Open App

<img width="1280" height="701" alt="image" src="https://github.com/easyparcel/nextgen-integration-doc/blob/6ed5f937fac29052efef52b6c997de2b1d457be3/Pictures/General1.png" />

---
## Steps to Switch
1. Go to the **Version** section.  
2. Toggle the switch from **Next Gen** to **Classic**.
3. Press the **Save button**
