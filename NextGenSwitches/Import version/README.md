# Import version migration
This folder is a guideline of how to migrate your EasyParcel Import version integration with your Existing Store from Classic version to NextGen version.

More details of nextGen you may refer [click here](https://github.com/easyparcel/nextgen-integration-doc/blob/fc48d953fb8081b09ffaadb9887a6ddab521bd1e/README.md)

If you have not set up the import version integration and would like to set it up, you may [click here](https://github.com/easyparcel/nextgen-integration-doc/tree/fc48d953fb8081b09ffaadb9887a6ddab521bd1e/Import).
