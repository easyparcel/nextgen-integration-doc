# Lazada Migration
To migrate your Classic EasyParcel import version integration with your Lazada Store to NextGen version. You need to follow this file for the setup steps.

### Navigation path:        
**Login your EasyParcel Account -> Integration -> Installed Ecommerce Apps -> Lazada Import -> Open App -> Select your Store**

You will be directed to this page. After that, click the **Order** button.

Click the re-authorize button, to re-authorize your Import version integration from Classic to NextGen.
