# Switching your Individual API Integration from NextGen to Classic Version
To become back the Classic User, you will need to ensure that your API key version is in Classic version. For the guideline that how to change API key version, you may refer to the steps below.

## Navigation Path
#### Login to EasyParcel website -> EasyConnect -> Installed Ecommerce Apps -> Individual API -> Open App

<img width="1280" height="708" alt="image" src="https://github.com/user-attachments/assets/d6b1f431-43c6-4915-a93e-a85864a72506" />

---

## Steps to Switch
1. Scroll to the **Version** section at the bottom.  
2. Toggle the switch to **Classic**.  
<img width="915" height="476" alt="Screenshot 2026-03-09 at 2 22 44 PM" src="https://github.com/user-attachments/assets/7ce05332-6a62-4999-92e3-905a15a92b91" />

### ✅ Done! You’ve now switched to **Classic**.
