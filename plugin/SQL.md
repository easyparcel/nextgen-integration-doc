# EasyParcel SQL NextGen Integration - Plugin Version

#### Step 2: Click 'EasyConnect'.
![EasyParcel SQL Step 1](https://github.com/easyparcel/nextgen-integration-doc/raw/main/Pictures/step1.png)

#### Step 3: Click 'Add Ecommerce App'. Find the 'EasyStore'. After that, click 'Install App'.

<img width="990" height="767" alt="sql11" src="https://github.com/user-attachments/assets/51245c0d-187c-4613-beee-d4834b85a86b" />

#### Step 14:
#### 01. Enter the Easy Parcel API key
#### 02. Enter the Sender Name (If you wanted to override the Company Profile Attention)
#### 03. Enter the Sender Mobile No (If you wanted to override the Company Profile Phone2)
#### 04. Click 1. Get Courier List…
#### 05. Click + to add Sync Program
#### 06. Select the Courier to use
#### 07. Click Check for Save
#### 08. Click Next
<img width="884" height="711" alt="sql12" src="https://github.com/user-attachments/assets/f4771d08-0eab-4d8c-9f80-a88125935fdd" />

#### Step 15:
#### 09. Select Document Type
#### 10. Select Delivery address Type
#### 11. Select Date Range
#### 12. Select other filter (if Any)
#### 13. Click “1. Get List…” to get the document list
#### 14. Tick & Enter the Weight (You can change the Courier too if require)
#### 15. Click “2. Get Rate…” to get estimate rate charges
#### 16. Click “3. Place Order…” if confirm
<img width="884" height="744" alt="sql13" src="https://github.com/user-attachments/assets/dabf0979-9c53-49e7-9173-9a9c97ee8118" />
